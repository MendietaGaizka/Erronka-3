# Promox Backup Server

Bi nodoko Proxmox kluster baten bidez gure azpiegitura birtualizatua kudeatzen dugu, non makina birtualak (VM) eta kontainerrak (CT) modu eraginkorrean martxan dauden. Segurtasun kopien kudeaketa hobetzeko, kluster horri Proxmox Backup Server bat gehitu diogu, eta horren bidez VM eta CT guztien kopiak modu zentralizatuan eta erregularrean gordetzen dira. Babes kopien plangintza eta politika zehatzak beste [dokumentu ](segurtasun-kopia-plana.md)batean jasota daude, non kopien maiztasuna, gordetako kopien iraupena eta berreskuratzeko prozedurak zehazten diren.&#x20;

Horrela, sistemaren segurtasuna eta datuen eskuragarritasuna bermatzen ditugu, ustekabeko gorabeheren aurrean babes sendoa izanik.
